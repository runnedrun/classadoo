Choose from four course levels

Introduction to Web Apps

This course requires no programming experience. Students learn the building blocks of a website and make an "About Me" site on their first day. With a Classadoo teacher's guidance, they will develop the techniques to bring their unique designs to life.

Creating with JavaScript

JavaScript is the magic behind interactive websites. Students program in JavaScript to add image slideshows, chat rooms, and even games to their web pages.

Advanced Web Development

This course takes JavaScript to the next level. Students learn to integrate all the tools they've learned so far. The will use the technology behind message boards, stores, and blogs to make their own fully-featured websites.

Web Development and Entrepreneurship

Students design, develop, and launch their own web apps in the real world! Think about popular web apps like Facebook, Craigslist, and Tumblr - students will learn to take an idea and turn it into the next big hit!
