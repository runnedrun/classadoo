We teach

Computer Science, Engineering and Design Thinking, through web development.

But we teach more than just concepts.

Classadoo was founded by engineers from top companies like Google and Twitter and educators from top schools like Olin College of Engineering and Nueva School. We create lessons which teach students to do the kind of our professionals do every day, through appropriately scaled hands on projects. After pilot programs in three schools, our instructors now spread their love of programming to schools and students accross the country.

Courses

Each course is 6 weeks long. Custom courses can be created for Classadoo in School users.

Fresh Start

For students with less than six months prior programming experience

Creating Webpages

From zero to web developer in six weeks. Learn how to use HTML, CSS and a little bit of Javascript to express yourself online. We'll create a blog, a news aggregator, music player and more - every hour long class produces a new website.

Jump Start

For students with at least six months programming experience, but little or no experience with web development.

Web Dev Express

The first half of this course covers the fundamentals of web development while the second half uses your existing programming knowledge as a launching point for more advanced javascript work.

Creator

For students who have taken a Shortcut or Fresh Start class, or have had 6 months of web development experience.

Dynamic Websites with Javascript

Bring your websites alive with image carousels, drag and drop menus, games and more. This course covers the design and implementation of Javascript interfaces on webpages.

Persistent Websites with Javascript

How does Facebook keep track of all the Likes for your photo? How does Google know your name and search preferences? Learn how to safely store and use user data to enhance your websites.

Master

For students who have taken both Creator classes.

Javascript and Data

Learn how to use Javascript to process, interpret and visualize data. We'll do textual analysis on books, numerical analysis on census data and much more.

Javascript and Design

Learn to use design and empathy work to create excellent interfaces for any user.

Javascript and Art

Learn to create interactive art, either as a standalone project or to enhance an existing site.

Entrepreneurship: Building a User Base

In partnership with Launch Camp, learn how to attract and retain users for your website.

Entrepreneurship: Making Money Online

In partnership with Launch Camp, learn how to market and sell a product online through your own website.
