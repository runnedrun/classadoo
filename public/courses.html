Courses are $25 per student, per hour long session. We offer two course formats:

Local Course

All students are together in one physical classroom, while a classadoo teacher teaches via a video call. This allows for more collaboration and socialization between students. Send an email to learn@classadoo.com to schedule a local class in your area!

Distributed Course

We offer this format when students are spread across multiple geographic areas. Each student logs into a secure, live video stream of the teacher, from anywhere. For safety, students do not see or hear other students, but any student can directly talk to the teacher at any time. Coming Summer 2016!
Most courses are available in both formats.

Web Apps with Javascript

 

Learn the basics of Javascript then immediately start building web apps relevant to your interests through Intro to Web Apps (Javascript) and the Create: Javascript series (level I-III). Pair these with the Create: Projects series, and you'll soon be ready to design, develop, and launch your own web apps through our Web Development and Entrepreneurship master course! Think about popular web apps such as Facebook, Reddit, and Tumblr - you could be the one to make the next big hit!

Creating with Scratch

 

Scratch is a fun way to learn how to code using simple blocks, customizable characters, colorful backdrops and more. While it is easy to learn, the possibilities and potential for complexity are endless. Take the Starter's Guide to Scratch and Create: Scratch to learn key coding concepts and work on projects that challenge the imagination and your problem solving skills! Great as a kicking off point for younger students.

Android

 

Know how to make web apps and excited for more? Take our master course Android Development and Entrepreneurship to learn how to code phone apps and submit one to the Play store! This is a master-level course, to be taken after a student has learned the ins and outs of web development.
